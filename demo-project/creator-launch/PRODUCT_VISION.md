# Product Vision — Creator Launch

## Scope Split

### In Scope for Tier 1 (launch — ship now)
- Channel trailer (60s)
- First 3 videos scripted, filmed, edited
- Channel art: banner, avatar, thumbnail template
- Upload schedule: 1 video/week
- Basic end screen + subscribe nudge

### In Scope for Tier 2 (after 1,000 subs)
- Podcast spin-off
- Community Discord server
- Shorts repurposing pipeline
- Collab videos with other creators

### Out of Scope for Now
- Merch store
- Paid course
- Multi-channel network deals
- Custom website
